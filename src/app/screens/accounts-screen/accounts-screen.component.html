Bookkeeping accounts is an application that let you store some accounts and setup a set of rules of transactions in order to check if a transaction between two accounts is legal or not. Pretty fancy, no?

In order to use this application, you need to upload a CSV with the accounts on which you want to set the transaction rules. The CSV should be in a two columns format having a comma ',' as a separator between the columns. First column represents an unique identifier and the second one is the account name. You can find a list of the valid accounts in Romania here

Valid accounts example:

...

378,Diferențe de preț la mărfuri
...

After you uploaded your accounts, you can now set some transaction rules. To do that, click on "Transaction Rules" button from below and the transaction rules screen will be prompted to you. There you can add rules between multiple accounts in order to easily see under what circumstances you are allowed to move assets from one account to another.

What remains to be done now is to go to the "Check Transactions Validity" section so that you can check where and from which accounts you can move your assets. As they say, keep your assets close and your transaction rules closer.
